Emotion Matrix Editor

Click a real LED once to turn it pink. Click it again to make it dim. Click a third time to turn it off.
The code panel updates automatically and outputs coordinates that can be pasted into your MicroPython animation code.
Matrix shape: 18 / 20*3 / 22*9 / 20*3 / 18 / 16. Hidden cells are not real LEDs.
