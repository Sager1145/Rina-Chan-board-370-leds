# WEBFIX 1.4.2-webfix

本版针对网页控制台做整理和稳定性修复。

## 修改内容

- 修复网页标题/副标题版本显示混乱：统一为 `1.4.2-webfix`。
- 移除页面中的旧入口：旧语音页、旧歌曲页、旧 flyAkari 页不再显示。
- Unity 语音 / 歌曲 / 影像统一保留在一个 `Unity媒体` 页面。
- 把滚动文字、自动同步、电池状态、表情部件随机按钮、表情部件共享保存列表改为初始 HTML 直接存在，减少加载后动态插入导致的页面跳动。
- 自定义表情页和表情部件页继续共用同一个保存列表。
- 自定义表情页、表情部件页、首页不再额外插入重复的 370 Matrix 预览卡。
- 已重新生成：
  - `webui/index.html`
  - `esp8258_bridge/data/webui_index.html.gz`
  - `esp8258_bridge/include/web_app_gz.h`

## 保留内容

- 370 LED 真实矩阵编辑器。
- 表情部件缩略图预览。
- Unity 媒体时间轴 370 Matrix 预览。
- 滚动文字页面。
- 原版二进制协议测试页面。
