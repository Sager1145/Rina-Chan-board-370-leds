# v1.5.3 保存表情管理更新

- 默认 faces 已按当前文件重命名，移除“默认 01”这类冗余前缀。
- 保存表情属性支持：默认、自定义表情、表情部件。
- 每个保存项包含锁定状态；锁定项不能删除。默认项仍强制禁止删除。
- 保存时提供“编号”和“名称”两个输入框，编号自动填充为下一个自定义编号。
- 默认表情编号和自定义/部件编号分开，默认编号显示为 `*01`、`*02`。
- 保存列表显示：编号、名称、锁定按钮、属性/自定义按钮、删除按钮、拖拽条。
- 默认表情与已保存表情都可以参与排序；排序支持鼠标和手指拖拽。
- 删除前面的自定义/部件表情后，后面的编号会自动前移。
- Web UI 会用 `saveFaces370Json|...` 同步完整排序列表到 Pico，使 B1/B2 和 A/M 循环顺序跟网页一致。
