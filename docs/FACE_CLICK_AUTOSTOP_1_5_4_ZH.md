# FACE_CLICK_AUTOSTOP_1_5_4

本版基于 1.5.3 保存表情管理继续修改：

- 保存表情列表显示列固定为：编号、名称、锁定按钮、自定义按钮、删除按钮、拖拽条。
- 编号按当前列表从上到下显示 1-99；默认表情行显示 `*` 前缀。
- 单击保存表情行即可立即载入，不再需要双击或再点“载入”。
- 切换页面时会自动停止当前滚动文字或 Unity 媒体播放。
- 选择其它保存表情或其它 Unity 媒体项目时会自动停止当前滚动/播放。
- 移除“播放/滚动中，页面控制已锁定；停止后才能切页、编辑或发送其它命令。”锁定逻辑与提示。
- 拖拽排序继续使用 Pointer Events，并加入 pointer capture 以提升鼠标/手指拖动稳定性。
