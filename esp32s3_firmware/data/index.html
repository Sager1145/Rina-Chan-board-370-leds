固件接口

LED 输出只发送 M370:<93 hex>；颜色、亮度、模式、暂停、按钮、ADC/status 等通过辅助 JSON 指令发送。

资源 / 系统

表情部件资源由 HTML 处理并合成为 M370；默认表情和用户保存表情都从同一个 /resources/saved_faces.json 读取，通过 /api/saved_faces 写回固件文件；type:"default" 项不可删除，但可重命名和排序。
